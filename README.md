# Master's Thesis Template

This is a $\LaTeX$ template for a FCUL Master's thesis written in English.

The structure of this template is loosely inspired by the [`templates_latex_fcul`](https://github.com/dpavot/templates_latex_fcul) GitHub repository.


## How To Use This Template

Clone this repo and edit the `*.tex` files in [chapters](chapters) as needed. If your thesis structure differs from the generic one in [template_output.pdf](template_output.pdf), edit [main.tex](main.tex) and add, remove, and/or rename the other `*.tex` files as needed.

Once you're ready to get a `.pdf` output, compile `main.tex`.

### Overleaf Users

Get a copy of the folders [chapters](chapters) and [images](images), and the files [main.tex](main.tex) and [refs.bib](refs.bib), and include them in a new Overleaf project. Edit the `*.tex` files as described above and compile your document from [main.tex](main.tex).

### VSCode Workspace Settings
To be able to compile your document in VSCode, include the settings provided in [settings.json](.vscode/settings.json) in your workspace `settings.json` file. If you cloned this repo as is, this is already taken care of.


## Document Norms

This template follows the official FCUL layout norms for the **2024/2025** academic year, which I have preserved in [normas_trabalho_final_2025.pdf](normas_trabalho_final_2025.pdf). It is your responsibility to verify whether these norms have been updated. As of date of the last commit, the official FCUL norms can be found [here](https://ciencias.ulisboa.pt/ensino/estudantes-atuais/informacoes-academicas/avaliacao-de-conhecimentos-e-certificacao/admissao-a-provas-academicas/2-ciclo) or [here](https://ciencias.ulisboa.pt/en/education/current-students/academic-information/knowledge-assessment-and-certification/admission-to-academic-examination/admission-academic-exams-2nd-cycle).

If you'd like to see this template updated, feel free to make a pull request.
